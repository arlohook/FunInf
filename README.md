# **Welcome to FunInf**

`FunInf` provides a set of simple functions for performing correlation and multiplicity adjusted inference on coefficients from functional response models generated by the
`pffr` function from the `refund` package.

---

## Installation

You can install the development version from GitHub:

```r
# install.packages("remotes")
remotes::install_github("arlohook/FunInf")
```
